# Experiment 10

# Aim
To implement a method that determines whether a graph contains Hamiltonian circuit that is a cycle that visits every vertex exactly once every step starting vertex.

# Theory
A Hamiltonian circuit is a cycle in a graph that visits every vertex exactly one time and finally returns back to the starting vertex. In graph theory, Hamiltonian circuits are very important because they help in solving many real life problems such as route planning, travelling salesman problem, network design, scheduling and optimization problems. A graph is said to contain a Hamiltonian circuit if it is possible to start from one vertex, travel through all remaining vertices exactly once, and come back to the initial vertex without repeating any vertex in between. Unlike Euler circuits which focus on visiting every edge exactly once, Hamiltonian circuits focus on visiting every vertex exactly once.
In this experiment, we implemented a program to determine all possible Hamiltonian circuits present in a graph. The graph is given as input in the form of an adjacency matrix. An adjacency matrix is a square matrix in which rows and columns represent vertices of the graph. If there is an edge between two vertices, the corresponding matrix element contains 1, otherwise it contains 0. This matrix representation makes it easy to check whether two vertices are connected or not during the execution of the algorithm.
The program uses the concept of backtracking to find Hamiltonian circuits. Backtracking is a recursive problem solving technique in which we build the solution step by step and remove choices whenever they do not satisfy the required condition. In this experiment, the algorithm starts from a selected starting vertex and tries to visit all other connected vertices one by one. While moving through the graph, it checks that no vertex is repeated in the current path. If a vertex is already visited, the algorithm ignores that vertex and tries another possible vertex. This process continues recursively until either all vertices are visited or no further valid move is possible.
When the length of the current path becomes equal to the total number of vertices, the program checks whether the last vertex is connected back to the starting vertex. If such an edge exists, then a Hamiltonian circuit is found. The complete path including the return edge is then stored in a list of Hamiltonian circuits. If the last vertex is not connected to the starting vertex, then the path is rejected because it does not form a complete cycle. After storing one valid circuit, the algorithm backtracks and searches for other possible circuits in the graph.
In the implementation, separate functions are used to improve readability and modularity of the code. One function is used to find Hamiltonian circuits using recursion and backtracking. Another function is used to calculate the positions of vertices for drawing the graph. Additional functions are used for displaying the graph and highlighting the Hamiltonian circuits. The graph visualization is done using matplotlib. Nodes are displayed using circles and edges are drawn using straight lines. The Hamiltonian circuit edges are highlighted in red colour so that the required cycle can be clearly identified from the remaining graph edges.
The program also displays all Hamiltonian circuits graphically in multiple subplots inside one window. Each subplot represents one Hamiltonian circuit separately. The title of every subplot contains the sequence of vertices in the circuit. This graphical representation helps in better understanding of how different Hamiltonian circuits are formed in the graph. Since all circuits are displayed individually, it becomes easy to compare different possible solutions visually.
The main advantage of this experiment is that it provides a clear understanding of recursive graph traversal and backtracking techniques. It also helps in understanding adjacency matrix representation, cycle detection, recursion, and graph visualization. Through this experiment, we can observe how a computer systematically checks all possible paths and identifies only those paths that satisfy the Hamiltonian circuit conditions.
Hence, this experiment successfully demonstrates the implementation of a method to determine and display all Hamiltonian circuits present in a graph using adjacency matrix representation and recursive backtracking technique.
